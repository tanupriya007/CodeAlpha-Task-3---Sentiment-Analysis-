# Sentiment Analysis

## Project Overview
This project is part of my AI Data Analytics Internship. It focuses on analyzing text data to determine the sentiment expressed in it using Natural Language Processing (NLP) and Machine Learning techniques.

** Objective **
- Preprocess text data
- Perform sentiment classification
- Build and evaluate a machine learning model
- Visualize the results

** Tools & Libraries**
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn

** Files**
- Sentiment_Analysis2.ipynb – Jupyter Notebook containing the complete implementation.
- Dataset not included due to GitHub upload size restrictions
 
** Results**
The model successfully classifies text into different sentiment categories after preprocessing and feature extraction.
